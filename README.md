# Conecta Rápido

**Conecta Rápido** es un juego de mesa clásico de estrategia para dos jugadores, donde el objetivo es ser el primero en formar una línea de cuatro fichas consecutivas, ya sea en horizontal, vertical o diagonal. Es una excelente manera de ejercitar la mente mientras te diviertes.

## Características

- Juego para dos jugadores.
- Tablero de 6 filas por 7 columnas.
- Detección automática de victorias y empates.
- Reinicio automático del juego tras cada partida.
- Validación de entradas para prevenir errores.

## Instalación

Para ejecutar **Conecta Rápido**, necesitarás tener instalado Python 3.

## Cómo Jugar

1. **Inicio del Juego**: El juego comienza con un tablero vacío. Los jugadores se turnan para colocar sus fichas en las columnas del tablero.

2. **Colocar Fichas**: Los jugadores deben elegir una columna donde desean colocar su ficha. La ficha caerá a la posición más baja disponible en esa columna.

3. **Objetivo**: El primer jugador en formar una línea de cuatro fichas consecutivas en cualquier dirección (horizontal, vertical o diagonal) gana la partida.

4. **Empate**: Si el tablero se llena sin que ningún jugador haya conseguido conectar cuatro fichas, el juego termina en empate.

5. **Reinicio**: Después de cada partida, el juego se reinicia automáticamente.
